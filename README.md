# IoT-Based-Humidity-and-Temperature-Monitoring-Using-Arduino-Uno

This project is an IoT-based environmental monitoring system that integrates the DHT11 sensor, Arduino Uno, and ESP8266-01 Wi-Fi module to enable automatic data collection, processing, and cloud transmission. The DHT11 sensor measures temperature (°C) and humidity (%), converting the readings into digital signals processed by the Arduino Uno. The processed data is then transmitted via serial communication to the ESP8266-01 module, which uploads the information to the ThingSpeak cloud platform over a Wi-Fi network. Through ThingSpeak’s web interface, the data is visualized in graphical form, allowing users to monitor temperature and humidity conditions in real time from any internet-connected device.
